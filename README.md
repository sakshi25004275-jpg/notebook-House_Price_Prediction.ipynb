# House Price Prediction using Linear Regression

## Problem Statement
This project predicts house prices using the Linear Regression algorithm based on features such as area, bedrooms, bathrooms, stories, parking, and other house-related attributes.

## Dataset
Housing Price Prediction Dataset

https://www.kaggle.com/datasets/harishkumardatalab/housing-price-prediction

## Methodology
- Data Loading
- Data Preprocessing
- Exploratory Data Analysis (EDA)
- Linear Regression Model
- Model Evaluation
